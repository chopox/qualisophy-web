# Qualisophy Web

A modern website for Qualisophy - Formación, built with Astro and Tailwind CSS.

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- **Astro** - Modern static site generator
- **Tailwind CSS** - Utility-first CSS framework
- **React** - Interactive components
- **TypeScript** - Type safety

## Project Structure

```
src/
├── components/     # Reusable components
├── layouts/        # Page layouts
├── pages/          # Website pages
└── styles/         # Global styles
```

## Development

The development server runs on `http://localhost:4321` with hot reload enabled.

## Deployment

This project is configured for deployment on Vercel with serverless functions.
