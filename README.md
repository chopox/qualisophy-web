# Qualisophy Web

A modern website for Qualisophy - Formación, built with Astro and Tailwind CSS.

## For developers: Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

### Tech Stack

- **Astro** - Modern static site generator
- **Tailwind CSS** - Utility-first CSS framework
- **React** - Interactive components
- **TypeScript** - Type safety

### Project Structure

```
src/
├── components/     # Reusable components
├── layouts/        # Page layouts
├── pages/          # Website pages
└── styles/         # Global styles
```

## Beginner's guide: How to run locally

### Prerequisites

Before you start, make sure you have the following installed on your computer:

- **Node.js:** Download and install it from [nodejs.org](https://nodejs.org/) (LTS version recommended).
- **Git:** Download and install it from [git-scm.com](https://git-scm.com/).

### Clone the Repository

Open your terminal (Command Prompt, PowerShell, or Terminal on Mac) and run the following command to download the project:

```bash
git clone https://github.com/Qualisophy/qualisophy-web.git
```

### Navigate to the folder

Enter the project folder you just downloaded:

```bash
cd qualisophy-web
```

## Install the package manager (pnpm)

```bash
npm install -g pnpm
```

## Install dependencies

Download all the necessary libraries for the website to work:

```bash
pnpm install
```

## Start the Website

Everything is ready! Run this command to start the local server:

```bash
pnpm dev
```

You will see a message like Local: http://localhost:4321/. Open that link in your web browser to see the website.

### Deployment

This project is configured for deployment on Vercel with serverless functions.
